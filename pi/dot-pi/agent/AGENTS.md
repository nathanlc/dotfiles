## Understanding request
- If you are not sure what the user wants, ask for clarification
- The user sometimes uses speech-to-text. If something looks incorrect, ask
user for clarification

## Writing code
- Always load the `code-writing` skill before writing or editing any code

## Security/cryptography
- If tasked to implement a security or cryptography related feature or task,
check with the user if a package should be used instead. If you have a good
recommendation (secure, up to date), you can recommend it to the user.
